# Week 1 - Day 1 README Content (Queries 1 to 40)

# Day 1 - SQL Basics, Filtering, Aggregations and Joins

## Objective

The objective of Day 1 is to understand the basics of SQL operations using the Employee, Department, and Project tables. The exercises focus on:

* Retrieving data
* Filtering records
* Sorting
* Aggregate functions
* Grouping
* Joins
* Subqueries

# Tables Used

## Employee Table

Contains employee details such as:

* Employee ID
* Employee Name
* Age
* Salary
* Department ID
* Hire Date

## Department Table

Contains department information:

* Department ID
* Department Name

## Project Table

Contains project details:

* Project ID
* Project Name
* Department ID

# Queries 1 to 40 Overview

## Queries 1–10 : Basic SELECT and Filtering Operations

These queries focus on retrieving employee records using:

* SELECT statements
* WHERE clause
* LIKE operator
* BETWEEN operator
* DISTINCT values
* ORDER BY clause

### Concepts Learned

* Displaying all records
* Fetching selected columns
* Filtering based on salary and department
* Pattern matching using LIKE
* Sorting data in ascending and descending order

## Queries 11–20 : Aggregate Functions and Grouping

These queries introduce aggregate operations such as:

* COUNT()
* MAX()
* MIN()
* AVG()
* GROUP BY
* HAVING

### Concepts Learned

* Counting employees
* Finding highest and lowest salary
* Calculating average salary
* Department-wise grouping
* Filtering grouped results using HAVING

## Queries 21–30 : Joins Between Tables

These queries demonstrate relationships between:

* Employee and Department tables
* Employee and Project tables

### Join Types Practiced

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* CROSS JOIN

### Concepts Learned

* Combining records from multiple tables
* Displaying department names with employees
* Displaying projects assigned to departments
* Understanding matching and non-matching records

## Queries 31–40 : Subqueries and Nested Queries

These queries involve nested SELECT statements.

### Concepts Learned

* Employees earning above average salary
* Employees assigned to projects
* Employees not assigned to projects
* Using IN and NOT IN
* EXISTS and NOT EXISTS
* Scalar and correlated subqueries
