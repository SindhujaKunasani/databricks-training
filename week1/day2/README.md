# Week 1 - Day 2 README Content (Queries 41 to 64)

# Day 2 - Advanced SQL Operations and Analytical Functions

## Objective

The objective of Day 2 is to learn advanced SQL concepts using analytical and window functions along with string and null handling operations.

The exercises focus on:

* Advanced joins
* Ranking functions
* Window functions
* NULL handling
* String functions
* Analytical queries

# Tables Used

## Employee Table

Stores employee details including:

* Employee ID
* Employee Name
* Salary
* Age
* Department ID
* Hire Date

## Department Table

Stores department details.

## Project Table

Stores project information.

## Queries 41–45 : Advanced Join Operations

These queries focus on:

* CROSS JOIN
* RIGHT JOIN
* DISTINCT operations
* Department-wise employee counting

### Concepts Learned

* Cartesian products
* Handling unmatched records
* Counting grouped data
* Identifying unique values

## Queries 46–50 : Sorting and Aggregate Analysis

These queries involve:

* Ordering employee data
* Finding highest and lowest salary employees
* Counting unique departments

### Concepts Learned

* Sorting using ORDER BY
* Using subqueries with aggregate functions
* Identifying top and bottom salary records
## Queries 51–57 : Window and Ranking Functions

These queries introduce analytical SQL functions.

### Functions Used

* RANK()
* DENSE_RANK()
* ROW_NUMBER()
* AVG() OVER()
* LAG()
* LEAD()

### Concepts Learned

* Ranking employees by salary
* Partitioning data by department
* Comparing current and previous salary values
* Sequential row numbering

## Queries 58–63 : NULL and String Handling Functions

These queries focus on:

* NULL checking
* COALESCE function
* Handling missing values

### Concepts Learned

* Identifying NULL records
* Replacing NULL values with defaults
* Improving data consistency
